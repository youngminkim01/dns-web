## 🎯 주요 기능

### 실시간 DNS 관리
```
사용자 → 웹 인터페이스 → Express 서버 → SSH → DNS 서버 (BIND9)
```

1. **조회**: 서버의 존 파일을 SSH로 읽어 파싱
2. **추가**: 존 파일에 레코드 추가 → 시리얼 증가 → BIND9 재로드
3. **수정**: 기존 레코드 수정 → 시리얼 증가 → BIND9 재로드
4. **삭제**: 레코드 제거 → 시리얼 증가 → BIND9 재로드

## 💡 기술적 하이라이트

### 1. SSH를 통한 안전한 원격 관리
```javascript
// 임시 파일 생성 → sudo로 이동 → 권한 설정
await executeSSHCommand(`echo '${content}' | sudo tee ${tempFile} > /dev/null`);
await executeSSHCommand(`sudo mv ${tempFile} ${zoneFile}`);
await executeSSHCommand(`sudo chown bind:bind ${zoneFile}`);
```

### 2. 정규식을 활용한 시리얼 번호 관리
```javascript
const serialRegex = /(\d{10})\s*;\s*Serial/;
const newSerial = currentSerial + 1;
content = content.replace(serialRegex, `${newSerial} ; Serial`);
```

### 3. 스마트한 존 파일 파싱
```javascript
// SOA, NS 레코드 제외
if (!['SOA', 'NS'].includes(recordType)) {
    // MX 레코드의 우선순위 추출
    if (recordType === 'MX') {
        record.priority = parseInt(parts[inIndex + 2]);
    }
}
```
